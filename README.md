# NBA API TS

REST API built with Node.js, TypeScript, Express, Prisma ORM and MySQL.

Project created to practice backend architecture patterns used in real-world applications.

---

## Stack

- Node.js
- TypeScript
- Express
- Prisma ORM
- MySQL
- Docker
- Zod Validation

---

## Features

- Create player
- List players
- Get player by ID
- Update player
- Delete player
- Pagination
- Dynamic filters
- Input validation
- Error handling middleware
- Repository Pattern

---

## Project Architecture

Controller

↓

Service

↓

Repository

↓

Prisma ORM

↓

MySQL Database
